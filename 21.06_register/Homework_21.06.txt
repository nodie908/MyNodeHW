Курс: Создание приложений на Node.js
Тема: Сетевые протоколы и HTTP

Задание
=======
Доработать код, созданный на занятии. 
Реализовать отдельное поведение для GET и POST запросов.

1. Для GET запросов добавить обработку url "/register", по которому отправлять файл "register.html".

2. В файле index.html добавить ссылку на страницу регистрации.

3. Файл register.html должен содержать форму регистрации нового пользователя и по кнопке Submit отправлять POST запрос на url "/users".

4. Обработчик POST запроса "/users" должен вытащить данные нового пользователя из тела запроса и дописать эти данные в файл users.json.
В качестве ответа сервер должен оправить клиенту html-код с автоматическим редиректом на "/users?name=%name%", где %name% - это имя нового созданного пользователя.

5. Как в html сделать автоматический редирект - найдите самостоятельно.

6. Модифицируйте обработчик GET запроса "/users" - он должен отдавать страницу users.html только в том случае, если пользователя в "name=" зарегистрирован, т.е. такое имя есть в users.json.
