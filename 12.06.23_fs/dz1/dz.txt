Перед выполнением домашнего задания полезно будет ознакомиться со статьёй «Работа с fs модулем Node.js»


1.	Написать код, который будет запрашивать у пользователя имя, фамилию и дату рождения, а также имя файла, в который эту информацию записать, и который записывает всю эту информацию в указанный файл.  


2.	Написать код, который считывает JSON файл такого вида:
{
  "count": 0
}
Вывести count в консоль, увеличить на один и перезаписать JSON файл.



3*. Написать программу - конвертер файлов из json в xml и обратно. Запускаться конвертер должен так:
node convert.js source.json destination.xml
или наоборот
node convert.js source.xml destination.json
Направление конвертации программа должна определять автоматически по расширению файлов.
