Курс: Создание приложений на Node.js
Тема: SQL + Telegram API

Задание 1
=========
Исследуйте код, созданный на занятии.

Задание 2
=========
Доделайте функционал бота для поиска событий по названию города и дате. Для тестирования не забудьте в config.ini указать токен своего бота.
Результат: 
 * пользователь заходит в чат с ботом, нажимает кнопку "Узнать о событиях",
 * отвечает на вопрос "В каком городе?"
 * отвечает на вопрос "В какой день?"
 * получает от бота несколько сообщений - в каждом сообщении вся информация о событиях в указанном городе и в указанную дату.

Задание 3
=========
Доделайте функционал бота для создания событий.
Результат:
 * пользователь заходит в чат с ботом, нажимает кнопку "Создать событие",
 * последовательно отвечает на все вопросы, в соответствии с полями таблицы Events,
 * пользователь получает сообщение, содержащее всю информацию о новом событии пользователя и вопрос "Всё верно?"
 * если пользователь отвечает "да",  в базе данных создаётся событие с указанными данными, а пользователю отправляется сообщение "Событие опубликовано".
